# 🛡️ World War - Official Client

Seja bem-vindo ao World War! Abaixo você encontra as opções para baixar o cliente e conectar ao nosso servidor.

---

## 🚀 Opção 1: Cliente Oficial (Recomendado)
Este cliente já vem configurado com todos os visuais modernos, montarias e IP fixo. Basta baixar, extrair e jogar!

*   **Download**: [Clique aqui para baixar via GitHub Releases](https://github.com/iflcosta/war-client/releases)
*   **Como usar**: Extraia a pasta e execute o arquivo `otclient_gl_x64.exe` ou `otclient_dx_x64.exe`.

---

## 🛠️ Opção 2: Cliente Clássico (Plano B)
Se você tiver problemas de performance com o cliente principal, utilize esta versão clássica da CipSoft com suporte a montarias.

### 1. Baixe o Cliente
*   **Download**: [Mateuzkl CipSoft 8.60 with Mounts](https://github.com/Mateuzkl/Client-cip-8.60-with-DLL-Mount/archive/refs/heads/main.zip)

### 2. Baixe o IP Changer
*   **Download**: [Otland IP Changer](https://static.otland.net/ipchanger.exe)

### 3. Configurações de Conexão
Abra o IP Changer e utilize as seguintes configurações:
*   **IP**: `189.46.133.165`
*   **Port**: `7171`
*   **Version**: `8.60`

---

## 🩺 Problemas Comuns
*   **Erro de DLL**: Certifique-se de ter o [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) instalado.
*   **Antivírus**: Alguns antivírus podem barrar o cliente. Adicione a pasta do jogo às exclusões se necessário.
*   **MacOS / Linux**: Utilize a **Opção 3** abaixo para jogar diretamente pelo navegador.

---

## 🌐 Opção 3: Jogue pelo Navegador (Mac/Linux/No-Download)
Ideal para usuários de Mac ou para quem quer testar o jogo sem baixar nada.

*   👉 **[ENTRAR NO JOGO VIA NAVEGADOR](https://webclient.otarchive.com/?gameData=https://wwar.servegame.com/worldwar_web_lite.zip)**

> [!IMPORTANT]
> Se for sua primeira vez jogando via navegador, você precisará autorizar a conexão segura.
> 1. Clique no link acima.
> 2. Se o jogo não conectar, abra [este link](https://wwar.servegame.com:8081) e clique em **"Avançado" -> "Prosseguir"**.
